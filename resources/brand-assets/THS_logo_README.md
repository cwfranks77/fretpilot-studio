# THS Monogram Logo (The Franks Standard)

This logo implements your spec: a tall T, a short H, and a tall S with a subtle casting shadow. Brand colors and type align with The Franks Standard.

## Files
- THS_logo_dark.svg — Gold monogram on dark background (#232323)
- THS_logo_gold.svg — Dark monogram on gold background (#FFD600)
- THS_icon_gold.svg — Square icon for avatars and favicons

## Brand Colors
- Gold: #FFD600
- Dark: #232323

## Typography
- Montserrat (Bold/Black). Fallback: Arial, sans-serif

## Usage
- Website header: THS_logo_dark.svg for dark header sections
- Social profile / favicon: THS_icon_gold.svg (export to PNG 512, 256, 128, 32)
- Print or merch: Use the SVGs for crisp scaling

## Export Tips
- For PNG exports, use sizes: 4096, 2048, 1024, 512, 256, 128, 64
- Keep the drop shadow; it adds depth without overpowering

## Variations Available
- Flat (no shadow)
- Outline/Stroke variants
- Monochrome (all gold or all dark)

If you want more alternates (e.g., stacked vertical THS, circular badge, embossed), tell me and I’ll add them.
