# Pre-Launch Checklist

## Android
- [ ] AAB uploaded to Play Console
- [ ] App content ratings completed
- [ ] In-app products approved
- [ ] Store listing (screenshots, icon, description)
- [ ] Billing permission present in AndroidManifest.xml

## iOS
- [ ] Developer Program active
- [ ] Certificates and profiles configured
- [ ] In-app purchases created and approved
- [ ] Screenshots and metadata ready
- [ ] Privacy policy URL live

## Windows
- [ ] Code signing certificate applied (optional)
- [ ] Installer tested on clean Windows machine

## Backend
- [ ] Vercel deployment live
- [ ] Environment variables configured
- [ ] Stripe webhook working
- [ ] BTCPay webhook working
- [ ] Dropship endpoints tested

## Security
- [ ] All secrets in secret manager (not committed)
- [ ] HTTPS enforced

## Marketing
- [ ] Website updated (pricing, features)
- [ ] Support email live
- [ ] Release notes drafted
