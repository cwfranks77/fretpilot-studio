# BTCPay Server Credentials

- BTCPAY_SERVER_URL: Your instance URL (e.g., https://btcpay.example.com)
- BTCPAY_STORE_ID: Found in Store → Settings → General
- BTCPAY_API_KEY: Account → API Keys (with invoice permissions)
- BTCPAY_WEBHOOK_SECRET: Arbitrary secret for webhook verification

Guide: See docs/BTCPAY_SERVER_SETUP.md
