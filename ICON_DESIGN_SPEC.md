# FretPilot Studio App Icon Design Specification

## Design Concept: Musical Theme

### Primary Design Elements
- **Guitar fretboard** as the central element
- **AI/Tech accent** (subtle circuit pattern or glow)
- **Color scheme**: Electric blue gradient (#00d4ff → #0066ff) with dark background
- **Style**: Modern, bold, professional

### Icon Variations Needed

#### 1. Main App Icon (512x512px)
**Description**: Guitar neck/fretboard view with glowing strings
- Dark background (#0a0a0f to #0b0d15 gradient)
- 6 strings visible (glowing electric blue)
- Frets visible with position markers
- Subtle AI circuit pattern in background
- "FP" monogram or small guitar pick in corner

#### 2. Simplified Version (for small sizes 48x48, 72x72)
**Description**: Abstract guitar pick or string motif
- Single bold guitar pick shape
- Gradient fill (blue to cyan)
- Dark border for contrast
- Minimal details for clarity at small size

### Color Palette
```
Primary Blue: #00d4ff
Secondary Blue: #0066ff
Dark Background: #0a0a0f
Accent Glow: rgba(0, 212, 255, 0.6)
Border/Outline: #1a1a2e
```

### Technical Requirements
- **Format**: PNG with transparency
- **Sizes needed**:
  - 512x512px (Google Play Store listing)
  - 192x192px (Android adaptive icon)
  - 144x144px
  - 96x96px
  - 72x72px
  - 48x48px
  - 1024x1024px (iOS App Store)

### Design Guidelines
1. **Safe area**: Keep important elements within 80% of canvas (avoid edges)
2. **Contrast**: Ensure icon is visible on both light and dark backgrounds
3. **Simplicity**: Avoid text (except subtle monogram)
4. **Recognition**: Should be instantly recognizable at 48x48px
5. **Brand consistency**: Match app's gradient blue theme

### Inspiration References
- Spotify (music + tech)
- Yousician (guitar learning)
- GarageBand (musical creation)
- But with **AI/futuristic tech aesthetic**

## Recommended: Concept A (Fretboard View)
This design:
- Instantly communicates "guitar learning"
- Modern with glowing tech aesthetic
- Scales well to small sizes
- Unique and memorable
- Matches app's blue gradient theme
